Adds a shortcode to display daily currency rate of USD, EUR and oil to Rouble.

== Description ==
- [vital_currency_rates] shortcode shows USD/RUB, EUR/RUB and Brent/RUB rates.
- Also displays change of currency according to previous day.

== Installation ==

1. Upload `vital-currency-rate` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use [vital_currency_rates] shortcode in content or text widget
4. Or use `<?php do_shortcode('vital_currency_rates'); ?>` in your templates
